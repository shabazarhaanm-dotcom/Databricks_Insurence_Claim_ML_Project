# Insurance Claims Automation Pipeline

## Overview

This project implements a scalable ETL pipeline following the **medallion architecture** to streamline and automate the insurance claims process. By leveraging **Machine Learning**, the system assists in validating and processing claims, aiming to improve accuracy and reduce resolution time for customers.

## Key Features

*   **Medallion Architecture**: Implements a multi-layered (Bronze, Silver, Gold) data flow to ensure data quality and reliability.
*   **ML-Powered Insights**: Utilizes machine learning models to assist in claim verification and fraud detection.
*   **Automated ETL Process**: Extracts, transforms, and loads data from various sources into a structured format for analysis.

   ## Tech Stack
   
   **Data Processing**: Apache Spark, SQL Server, Lakebase, AWS S3, Kinisis
   **Machine Learning**: Scikit-learn, MLflow, Mosaic AI
   **Orchestration**: Databricks
   **Cloud Provider**: AWS
